# Analysis sandbox

To run the Jupyter notebook:

**From project root** (`rent-history/`):
   ```bash
   source .venv/bin/activate
   jupyter notebook src/analysis/sandbox.ipynb
   ```

If the venv doesn’t exist yet (from project root):
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r src/analysis/requirements.txt
```
